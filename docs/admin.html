Ρύθμιση εγγραφών (μία φορά)

Οι αλλαγές αποθηκεύονται με commit στο αποθετήριο μέσω του GitHub API. Χρειάζεται ένα fine-grained personal access token μόνο για αυτό το repo με δικαίωμα Contents: Read & write (GitHub → Settings → Developer settings → Fine-grained tokens). Μένει αποθηκευμένο μόνο σε αυτόν τον browser.

Συμβάντα

Οι αλλαγές εφαρμόζονται αμέσως στη σελίδα και στις ειδοποιήσεις από την επόμενη εκτέλεση. Ό,τι αφαιρεθεί δεν ξαναεμφανίζεται ούτε ξαναειδοποιεί.
